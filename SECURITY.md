# Security Policy

## Versi yang Didukung

Karena project ini adalah static site single-file (demo front-end), hanya versi terbaru di branch `main` yang menerima patch keamanan.

| Versi | Didukung |
| --- | --- |
| `main` (terbaru) | ✅ |
| versi lama / tag rilis | ❌ |

## Melaporkan Vulnerability

⚠️ **Penting:** Project ini adalah demo front-end tanpa backend nyata. Sistem login/register yang ada hanya menyimpan data di memory browser (tidak persisten, tidak terenkripsi, dan **tidak boleh digunakan untuk data sensitif/produksi**).

Jika kamu menemukan celah keamanan (misalnya XSS, injection pada form, atau eksploitasi DOM), mohon laporkan dengan cara:

1. **Jangan** buka issue publik untuk vulnerability yang berisiko tinggi
2. Kirim laporan melalui **GitHub Security Advisories** (tab *Security* → *Report a vulnerability*) di repo ini, atau
3. Hubungi maintainer langsung melalui email/kontak yang tercantum di profil GitHub

Sertakan dalam laporan:
- Deskripsi singkat vulnerability
- Langkah reproduksi
- Dampak potensial
- (Opsional) saran perbaikan

## Apa yang Bisa Diharapkan

- Konfirmasi penerimaan laporan dalam beberapa hari
- Update status investigasi secara berkala
- Credit di catatan rilis (jika diinginkan) setelah perbaikan dirilis

Terima kasih sudah membantu menjaga keamanan project ini! 🔒
